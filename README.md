## PR Reviewer Assignment Service

Учебный микросервис на FastAPI для автоматического назначения ревьюеров на Pull Request’ы внутри команды.
Сервис создаёт команды, пользователей и автоматически выбирает двух активных ревьюеров из команды автора PR.

#### Как запустить

#### Через Docker

В корне проекта выполните:
```
  docker compose up --build
```
После запуска сервис доступен по адресу:

#### http://localhost:8080

⸻

### Основные возможности
	•	создание пользователей
	•	создание команд и привязка участников
	•	автоматическое назначение ревьюеров (2 активных пользователя из команды автора)
	•	переназначение ревьюера
	•	завершение PR (merge)
	•	получение списка PR, назначенных конкретному пользователю

⸻

## Технологии
	•	Python 3.10
	•	FastAPI
	•	SQLAlchemy
	•	PostgreSQL
	•	Alembic
	•	Docker / Docker Compose

## Переменные окружения

Добавьте файл .env:
```
DB_HOST=postgres
DB_PORT=5432
DB_USER=postgres
DB_PASS=postgres
DB_NAME=avito_backend
```

## Документация API

После запуска сервиса доступна интерактивная документация FastAPI:

#### Swagger UI:
http://localhost:8080/docs

Здесь можно:
```
	•	просматривать все доступные эндпоинты,
	•	отправлять тестовые запросы прямо из браузера,
	•	видеть структуру запросов/ответов,
	•	быстро проверять работу сервиса без Postman.
```
